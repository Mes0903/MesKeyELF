# Mes Key ELF

This is an program that can help you send the keyboard and mouse signal to the background window.

# How to Build

```bash
git clone https://github.com/Mes0903/MesKeyELF.git
cd MesKeyELF
mkdir build && cd build
cmake ..
cmake --build .
```
